# Frontend

Placeholder for the future React/Vite frontend.

Planned areas:

- `app` - root app shell, styling, routing if needed
- `features/test-panel` - main operator workflow
- `features/settings` - template path, layout profile, and release settings
- `integrations/tauri` - typed Tauri bridge
- `shared/components` - reusable buttons, sections, dialogs, and status widgets
- `shared/lib` - formatting, state helpers, and frontend validation

The first UI milestone is a visual parity shell that keeps the legacy panel's operator flow.
