# v0.2.15 Corrective Automation, Changeover, and Dynamic Identity Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver monotonic backend-owned CSV readiness, strict STEP71/STEP72 burn-in sequencing, a one-time STEP42 Changeover Teams card after STEP41, and a schema-v2 add/rename identity catalog with floor/admin roles and editable comboboxes.

**Architecture:** Rust becomes the authority for scan readiness and repeats that preflight before all processing; React derives countdowns from absolute backend timestamps. The existing notification queue delivers a receipt-backed Changeover event without touching the shift log. Floor settings retain schema 1 until a dynamic identity is created, then atomically move to schema 2 with immutable backend-generated IDs and role-aware UI filtering.

**Tech Stack:** Tauri 2, Rust 2021 (`serde`, `serde_json`, existing `chrono`/`regex`/`walkdir`), React 19, TypeScript 6, Vite 8, Tailwind CSS, Bun, Vitest, Testing Library.

```rust
const DEFAULT_TASK_DURATION_SECONDS: u64 = 180;
const TRANSFORMER_DURATION_SECONDS: u64 = 60;
const SYSTEM_BURN_IN_SOAK_SECONDS: u64 = 7_200;
const SYSTEM_BURN_IN_CAPTURE_SECONDS: u64 = 60;

fn duration_ms(seconds: u64) -> u64 {
    seconds.saturating_mul(1_000)
}

fn task_nominal_duration_seconds(task: &tasks::AutomationTask) -> u64 {
    match task.kind {
        tasks::TaskKind::Transformer { .. } => TRANSFORMER_DURATION_SECONDS,
        tasks::TaskKind::SystemBurnIn => {
            SYSTEM_BURN_IN_SOAK_SECONDS + SYSTEM_BURN_IN_CAPTURE_SECONDS
        }
        _ => DEFAULT_TASK_DURATION_SECONDS,
    }
}

fn generic_readiness(
    task: &tasks::AutomationTask,
    csv_match: &TaskCsvMatch,
    now_ms: u64,
) -> TaskReadiness {
    let nominal = task_nominal_duration_seconds(task);
    let Some(start_ms) = csv_match.created_ms else {
        return TaskReadiness {
            process_ready: false,
            wait_phase: TaskWaitPhase::AwaitingCsv,
            phase_deadline_ms: None,
            pending_duration_seconds: nominal,
            nominal_duration_seconds: nominal,
        };
    };
    let deadline = start_ms.saturating_add(duration_ms(nominal));
    let (process_ready, wait_phase) = if now_ms < deadline {
        (false, TaskWaitPhase::Timing)
    } else if !csv_match.processable {
        (false, TaskWaitPhase::WaitingUnlock)
    } else {
        (true, TaskWaitPhase::Ready)
    };
    TaskReadiness {
        process_ready,
        wait_phase,
        phase_deadline_ms: Some(deadline),
        pending_duration_seconds: 0,
        nominal_duration_seconds: nominal,
    }
}

fn burn_in_readiness(csv_index: &CsvScanIndex, now_ms: u64) -> TaskReadiness {
    let nominal = SYSTEM_BURN_IN_SOAK_SECONDS + SYSTEM_BURN_IN_CAPTURE_SECONDS;
    let Some(step71) = csv_index.latest_for_step(71) else {
        return TaskReadiness { process_ready: false, wait_phase: TaskWaitPhase::AwaitingCsv,
            phase_deadline_ms: None, pending_duration_seconds: nominal,
            nominal_duration_seconds: nominal };
    };
    let Some(step71_start) = step71.created_ms else {
        return TaskReadiness { process_ready: false, wait_phase: TaskWaitPhase::AwaitingCsv,
            phase_deadline_ms: None, pending_duration_seconds: nominal,
            nominal_duration_seconds: nominal };
    };
    let soak_deadline = step71_start.saturating_add(duration_ms(SYSTEM_BURN_IN_SOAK_SECONDS));
    let capture = csv_index.latest_for_step_fragments(
        72,
        &["SYSTEM_ACCURACY_TEST_DATA_AVG".to_string()],
    );
    let Some(capture) = capture else {
        return TaskReadiness {
            process_ready: false,
            wait_phase: if now_ms < soak_deadline { TaskWaitPhase::Soaking } else { TaskWaitPhase::WaitingStep72 },
            phase_deadline_ms: (now_ms < soak_deadline).then_some(soak_deadline),
            pending_duration_seconds: SYSTEM_BURN_IN_CAPTURE_SECONDS,
            nominal_duration_seconds: nominal,
        };
    };
    let Some(capture_start) = capture.created_ms else {
        return TaskReadiness { process_ready: false, wait_phase: TaskWaitPhase::WaitingStep72,
            phase_deadline_ms: None, pending_duration_seconds: SYSTEM_BURN_IN_CAPTURE_SECONDS,
            nominal_duration_seconds: nominal };
    };
    let capture_deadline = capture_start.saturating_add(duration_ms(SYSTEM_BURN_IN_CAPTURE_SECONDS));
    let ready_at = soak_deadline.max(capture_deadline);
    let wait_phase = if now_ms < ready_at {
        if soak_deadline >= capture_deadline { TaskWaitPhase::Soaking } else { TaskWaitPhase::Capturing }
    } else if !capture.readable {
        TaskWaitPhase::WaitingUnlock
    } else {
        TaskWaitPhase::Ready
    };
    TaskReadiness { process_ready: wait_phase == TaskWaitPhase::Ready, wait_phase,
        phase_deadline_ms: Some(ready_at), pending_duration_seconds: 0,
        nominal_duration_seconds: nominal }
}

fn evaluate_task_readiness(
    task: &tasks::AutomationTask,
    csv_index: &CsvScanIndex,
    csv_match: &TaskCsvMatch,
    now_ms: u64,
) -> TaskReadiness {
    if task.kind == tasks::TaskKind::SystemBurnIn {
        burn_in_readiness(csv_index, now_ms)
    } else {
        generic_readiness(task, csv_match, now_ms)
    }
}
```

```rust
fn readiness_wait_message(task: &tasks::AutomationTask, readiness: TaskReadiness) -> String {
    match readiness.wait_phase {
        TaskWaitPhase::AwaitingCsv => format!("Waiting for {} CSV.", task.label),
        TaskWaitPhase::Timing => format!("Waiting for {} CSV timer.", task.label),
        TaskWaitPhase::Soaking => "System Burn-In STEP71 soak is still running.".to_string(),
        TaskWaitPhase::WaitingStep72 => {

        TaskWaitPhase::WaitingUnlock => "Required CSV is still locked or unreadable.".to_string(),
        TaskWaitPhase::Ready => "Ready.".to_string(),
    }
}

fn process_preflight(
    task: &tasks::AutomationTask,
    profile: &ReportLayoutProfile,
    unit_folder: &Path,
    now_ms: u64,
) -> Result<Option<TaskProcessResult>, AutomationError> {
    let csv_index = CsvScanIndex::scan(unit_folder);
    let csv_match = task_csv_match(task, profile, &csv_index);
    let readiness = evaluate_task_readiness(task, &csv_index, &csv_match, now_ms);
    if readiness.process_ready {
        return Ok(None);
    }
    Ok(Some(TaskProcessResult {
        task_id: task.id.clone(),
        state: "waiting".to_string(),
        code: 2,
        message: readiness_wait_message(task, readiness),
        log: vec![csv_match.reason],
        report_path: None,
        print_report_path: None,
        failure: None,
        source_csv_path: csv_match.path.map(|path| path.display().to_string()),
        csv_fingerprint: None,
    }))
}
```

```tsx
  entries: StationCatalogEntry[];
  value: string;
  disabled?: boolean;
  allowCreate?: boolean;
  onValueChange: (value: string) => void;
  onSelect: (entry: StationCatalogEntry) => void;
  onCreate: (name: string, role: StationRole) => void;
};

export function IdentityCombobox(props: IdentityComboboxProps) {
  const [open, setOpen] = useState(false);
  const [activeIndex, setActiveIndex] = useState(0);
  const query = props.value.trim().toLocaleLowerCase();
  const matches = props.entries.filter((entry) => entry.name.toLocaleLowerCase().includes(query));
  const exact = props.entries.some((entry) => entry.name.toLocaleLowerCase() === query);
  const options = [
    ...matches.map((entry) => ({ key: entry.id, label: entry.name, run: () => props.onSelect(entry) })),
    ...(!props.allowCreate || !query || exact ? [] : [
      { key: "create-floor", label: `Add ${props.value.trim()} as Floor Station`, run: () => props.onCreate(props.value.trim(), "floor") },
      { key: "create-admin", label: `Add ${props.value.trim()} as Admin Identity`, run: () => props.onCreate(props.value.trim(), "admin") },
    ]),
  ];
  const listId = `${props.label.toLowerCase().replaceAll(" ", "-")}-listbox`;
  const choose = (index: number) => { options[index]?.run(); setOpen(false); };
  return <label className="block text-[8pt] font-semibold text-[#d8d2c8]">
    {props.label}
    <input role="combobox" aria-label={props.label} aria-expanded={open}
      aria-controls={listId} aria-activedescendant={open ? `${listId}-${activeIndex}` : undefined}
      value={props.value} disabled={props.disabled}
      onFocus={() => setOpen(true)}
      onChange={(event) => { props.onValueChange(event.target.value); setOpen(true); setActiveIndex(0); }}
      onKeyDown={(event) => {
        if (event.key === "ArrowDown") { event.preventDefault(); setOpen(true); setActiveIndex((index) => Math.min(index + 1, options.length - 1)); }
        if (event.key === "ArrowUp") { event.preventDefault(); setActiveIndex((index) => Math.max(0, index - 1)); }
        if (event.key === "Enter" && open) { event.preventDefault(); choose(activeIndex); }
        if (event.key === "Escape") setOpen(false);
      }} />
    {open && options.length > 0 ? <div id={listId} role="listbox">
      {options.map((option, index) => <button key={option.key} id={`${listId}-${index}`}
        type="button" role="option" aria-selected={index === activeIndex}
        onMouseDown={(event) => event.preventDefault()} onClick={() => choose(index)}>{option.label}</button>)}
    </div> : null}
  </label>;
}
```

Use the repository's existing classes on the input/list/buttons while preserving these roles and keyboard handlers.
