# PDU Data Automation v0.2.1

This hotfix grants the Tauri updater plugin permissions needed for the frontend updater flow.

## Changes

- Add `updater:default` to the main window capability.
- Show the updater retry button when an updater check enters the error state.
- Keep the `v0.2.0` startup and Browse performance changes.

## Important Note

The installed `v0.1.0` and `v0.2.0` builds cannot start the in-app updater because their Tauri capability files do not grant updater permissions. Install `v0.2.1` manually, then use a newer release to smoke test the in-app updater path.

## Validation

- `node scripts\run-bun.mjs run lint`
- `node scripts\run-bun.mjs run test`
- `node scripts\run-bun.mjs run build`
