# PDU Data Automation v0.2.0

This release is intended to test the updater path from the installed `v0.1.0` pilot.

## Changes

- Detect completed CSV-backed steps immediately after Browse, before slower report setup work finishes.
- Reduce repeated report discovery and avoid repairing untouched existing reports during folder setup.
- Delay the first automatic updater check by 15 seconds after startup.
- Remove unused DM Sans font imports and package dependency.
- Add lightweight startup timing logs for frontend load, React mount, backend status, layout profile load, and updater check completion.

## Validation

- `node scripts\run-bun.mjs run lint`
- `node scripts\run-bun.mjs run test`
- `node scripts\run-bun.mjs run build`
- `cargo fmt --manifest-path backend\Cargo.toml --check`
- `cargo check --manifest-path backend\Cargo.toml`
- `cargo test --manifest-path backend\Cargo.toml`
