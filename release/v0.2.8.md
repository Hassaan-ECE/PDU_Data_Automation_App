# PDU Data Automation v0.2.8

This release replaces the Start-time setup modal with inline unit selection and inline Transformer SN setup/save behavior.

## Changes

- Add inline unit selection with a read-only selected-unit field and `...` browse button.
- Remove startup latest-unit auto-selection from the frontend.
- Add inline Transformer SN entry with save-on-Start, save-on-blur, Enter save, and an icon-only save button.
- Keep Start-time setup writing through `setup_unit_folder_with_transformer_sn` when Transformer SN is supplied.
- Add `save_transformer_sn` for later Transformer SN edits and preserve numeric-looking values such as `000123` as text.
- Block report opening when Transformer SN is missing or unsaved, including failure-card report opening.
- Add explicit current-step follow controls: `Start`, `Resume`, `Follow Step`, and `Current Step`.
- Disable current-step follow mode when the operator manually scrolls or expands/collapses sections.
- Start automatic updater checks only after startup readiness, then after a short post-ready delay.
